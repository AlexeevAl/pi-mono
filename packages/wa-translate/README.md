# WhatsApp Translator Agent 🇮🇱🇷🇺

Интеллектуальный агент для автоматического перевода сообщений в WhatsApp. Позволяет бесшовно общаться с ивритоязычными и англоязычными контактами, не выходя из мессенджера.

## ✨ Основные функции

- **Автоматический перевод**: Переводит входящие сообщения на русский, а ваши исходящие — на выбранный целевой язык.
- **Гендерная адаптация**: Бот понимает, что вы — мужчина, и использует правильные мужские формы слов при переводе на иврит.
- **Контекстная память**: Помнит последние 5 сообщений в каждом чате для сохранения нити разговора.
- **Смарт-активация**: Автоматически включается, если собеседник прислал иностранное сообщение.
- **Гибкое управление**: Включение и выключение перевода командами прямо в чате.

## 🛠 Команды в чате

| Команда | Описание |
|---------|----------|
| `/translate on he` | Включить перевод на иврит |
| `/translate on en` | Включить перевод на английский |
| `/translate off` | Выключить перевод в этом чате |
| `/lang [код]` | Сменить язык (например, `/lang he`, `/lang en`) |

## 🚀 Быстрый старт

1. **Настройка**: Переименуйте `.env.example` в `.env` и вставьте ваш `OPENAI_API_KEY`.
2. **Установка**:
   ```bash
   pnpm install
   ```
3. **Запуск**:
   ```bash
   pnpm start
   ```
4. **Авторизация**: Отсканируйте появившийся в терминале QR-код через WhatsApp (Связанные устройства).

## ⚙️ Технологии

- **Node.js & TypeScript**
- **@whiskeysockets/baileys**: Синхронизация с WhatsApp.
- **pi-ai**: Интеграция с LLM (используется модель `gpt-5.4-nano`).
- **File Storage**: Настройки чатов сохраняются в `history.json`.
