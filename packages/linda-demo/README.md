# LINDA OS Clinic Presentation

Статическая презентация для продажи LINDA OS владельцам премиум-клиник.

## Как открыть

Откройте `index.html` в браузере.

Для локального сервера:

```bash
python -m http.server 5179 --bind 127.0.0.1
```

Затем открыть:

```text
http://127.0.0.1:5179/
```

## Что показывает

- расчет денег в текущей базе пациентов;
- сравнение текущего процесса и процесса с LINDA OS;
- запуск возврата первых 50 пациентов;
- диалог с пациентом;
- персональную страницу рекомендации;
- экран врача перед приемом;
- поствизитный возврат;
- экран владельца с ответами, записями и ожидаемым доходом.

## Языки

Переключатель `RU / EN / HE` меняет язык интерфейса и автопрезентации.

Для иврита включается направление текста справа налево (`rtl`).

## Рынки и валюта

В калькуляторе доступны 4 рынка:

- Израиль;
- США;
- Европа;
- Россия.

Выбор рынка меняет валюту и средний чек. Остальные значения можно менять вручную:

- пациентов в базе;
- неактивных пациентов;
- процент возврата;
- средний чек.

## Ограничения

Презентация не использует реальные API, мессенджеры, модель, базу данных или пациентские данные. Все сценарии и цифры демонстрационные.
